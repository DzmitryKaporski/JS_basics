(The first task) 1. Напишите функцию, которая принимает от одного до трех параметров, и возвращает их сумму. Типы данных для параметров: число или строка.
(The second task) 2. Напишите функцию, которая в качестве параметров принимает три строчных значения и возвращает сумму строк в алфавитном порядке, разделенных пробелом. Предполагается, что все значения введены на одном языке.
(The third task) 3. Напишите функцию, которая возвращает подпись к электронному письму в стиле "С уважением, {Имя Пользователя}". Имя пользователя передается в качестве параметра. Если этот параметр не передается, функция должна запросить его у пользователя при помощи prompt.
( The fourth task) 4. Напишите функцию, которая принимает в качестве параметров два положительных целых числа, и возвращает первое число возведенное в степень, равную второму параметру. В случае, если второй параметр не передан, функция возвращает первый параметр возведенный в квадрат.
