I. Создайте объект с тремя свойствами.
II. Добавьте к нему два новых свойства
III. Добавьте к объекту два метода, первый использует в качестве параметров внешние данные, второй - имеющиеся свойства объекта.
IV. Создайте функцию, которая выводит в консоль свойства и значения объекта в виде строки - "property: value."
V. Создайте функцию, которая определяет, является ли объект многоуровневым, другими словами, обладает ли объект свойствами, которые тоже представляют собой объекты. В качестве параметра функция принимает объект, и в случае, если объект многоуровневый - возвращает true, если нет - false. Случай, при котором свойство имеет значение null, также должен обрабатываться корректно.
